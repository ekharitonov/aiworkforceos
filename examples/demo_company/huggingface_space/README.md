# AI Workforce OS - Mini Content Agency

A working prototype of an AI-powered content agency demonstrating autonomous AI agent collaboration.

## Features

- **5 AI Agents** working together autonomously
- **2 Workflows**: Article Creation & Social Media Campaigns
- **100% Open Source** models from HuggingFace
- **Free to use** with HuggingFace Inference API

## AI Team

| Agent | Role |
|-------|------|
| Research Analyst | Gathers information and data |
| Content Writer | Creates articles and content |
| SEO Specialist | Optimizes for search engines |
| Social Media Manager | Creates social posts |
| QA Checker | Quality assurance |

## How to Use

1. Get your free [HuggingFace API Key](https://huggingface.co/settings/tokens)
2. Enter it in the Setup section
3. Select a model (Qwen2.5-7B recommended)
4. Click "Initialize Agency"
5. Try the workflows!

## Models

All models from [AI Workforce OS Collection](https://huggingface.co/collections/ekharitonov/ai-workforce-os-recommended-models):

- Qwen2.5-3B (Fast)
- Qwen2.5-7B (Recommended)
- Qwen2.5-14B (High Quality)
- Qwen2.5-Coder-32B (Best Quality)
